# 缩放移动 html 元素

## cssSelector

css 合法选择器

## options

```javascript
    parentWidth = document.body.clientWidth, // 父元素宽度
    parentHeight = document.body.clientHeight, // 父元素高度
    defaultScale = 1, // 默认缩放比例
    minScale = 0, // 最小缩放比例
    maxScale = 0, // 最大缩放比例
    limit = true, // 是否限制移动超出父级容器
    relateY = 0, // 纵向偏移量
```
